Để thuận tiện cho việc chấm điểm và rà soát logic, dự án được phân rã thành 3 khu vực độc lập dưới đây. Người đánh giá bài test có thể truy cập trực tiếp vào từng khu vực để xem mã nguồn và tài liệu hướng dẫn chi tiết:

### 1. Phần Trả lời câu hỏi Logic

- **Nội dung:** Lời giải cho các câu hỏi tư duy logic.
- **Đường dẫn:** Xem chi tiết tại file [**`logic-test.md`**](./logic-test.md) ngay tại thư mục gốc này.

### 2. Bài 02 — Chuẩn hóa dữ liệu bồi thường (Claims Data Cleanup)

- **Nội dung:** Module quét lỗi dữ liệu thô, chuẩn hóa tên/ngày tháng/mã tiền tệ, khử trùng lặp và xuất báo cáo thống kê.
- **Đường dẫn:** Truy cập thư mục [**`./02-Claims-Data-Cleanup-&-Report/`**](./02-Claims-Data-Cleanup-&-Report/)
- **Hướng dẫn:** Truy cập file [**`./02-Claims-Data-Cleanup-&-Report/README.md`**](./02-Claims-Data-Cleanup-&-Report/README.md)

### 3. Bài 06 — Tính toán quyền lợi bảo hiểm (Policy Benefits Calculator)

- **Nội dung:** Công cụ giả lập dịch vụ tự động, đẩy dòng tiền qua mô hình thác đổ để tính dồn tích hạn mức năm, trần lần khám, khấu trừ đầu kỳ và đồng chi trả.
- **Đường dẫn:** Truy cập thư mục [**`./06-Policy-Benefits-Calculator/`**](./06-Policy-Benefits-Calculator/)
- **Hướng dẫn:** Truy cập file [**`./06-Policy-Benefits-Calculator/README.md`**](./06-Policy-Benefits-Calculator/README.md)
